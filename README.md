FuzzyInput
=====

Prefix tree style fuzzy spell checker including auto-complete and auto-correction

### Insertion and Deletion Tolerance
    > neccesary
    necessary
### Auto-Complete
    > mot
    mother
### Keyboard Vicinity ([Gesture][crz]) 
    > moyjer
    mother
